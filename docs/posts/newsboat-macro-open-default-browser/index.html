Newsboat Macro Open Default Browser

May 9, 2021 #TIL 

My newsboat is configured to use w3m (a textonly browser) as default browser

browser "w3m %u"

from times to times you enconter some pages that does not look good on text-only browser , this config defines a macro to open the default browser by pressing ,m (COMMA + lowercase m)

macro m set browser "xdg-open %u"; open-in-browser ; set browser "w3m %u"
